About Me

My name is Tim Bedsole and I'm a full-stack web developer living in Florida. I love getting my hands dirty with code and building amazing applications from scratch. I recently received my full-stack web development certification from UCF, where I not only learned web development, but how to work in a team setting and how to really think like a developer. Working closely with likeminded individuals amplifies my passion for web development and keeps me engaged. I'm always trying to think of ways to do something better, and I live for that lightbulb moment.

Projects

I take pride on all of my work. Here's a few projects I've worked on.

Go Bunco

Go Bunco is a project that I am still currently working on. The purpose of this webapp is to bring the game of Bunco into the modern age by keeping track of scores in real time. This is a full-stack application that utilizes the Socket.io library.

DevDen

DevDen was a group project that was created within one week. It's a website for developers to find others who would like to colaborate. I was fortunate to work with a great team and we ended up with something we were all proud of. My role in creating DevDen was mostly back-end work on the database. We used MySQL.

Street Fighter Hangman

This Street Fighter themed Hangman game is in vanilla JavaScript. It was the first JavaScript homework I did in the boot camp and it's also one of the projects that I am most proud of. I learned a lot working on this and had a ton of fun, too.

World Warrior Finder

World Warrior Finder is another Street Fightered themed webapp. This was completed as one of my homework assignments in the boot camp. It utilizes express routing and algorithms to determine your best match.

Beer!

Beer! Is a simple webapp I created when tinkering around with translating user input to a database. I used MySQL.

GifTastic

GifTastic was an assignment in the boot camp that was meant to expose us to using APIs. This webapp uses the Giphy API. Once a user clicks a button, the Giphy API is searched with the value of that button as the parameter and GIFs of that search are rendered to the screen.
